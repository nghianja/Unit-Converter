Description

In this stage, you will write a program that can convert a value with any measure of length from the list below, using the metric system.

The conversion should follow these rules:

  1. One meter should, of course, consist of 1 meter.
  2. One kilometer should consist of 1000 meters.
  3. One centimeter should consist of 0.01 meters.
  4. One millimeter should consist of 0.001 meters.
  5. One mile should consist of 1609.35 meters.
  6. One yard should consist of 0.9144 meters.
  7. One foot should consist of 0.3048 meters.
  8. One inch should consist of 0.0254 meters.

The user should input a number and a type of measurement. These types of measurements should be allowed:

  1. For meters, the user can input "m", "meter", or "meters".
  2. For kilometers, the user can input "km", "kilometer", or "kilometers".
  3. For centimeters, the user can input "cm", "centimeter", or "centimeters".
  4. For millimeters, the user can input "mm", "millimeter", or "millimeters".
  5. For miles, the user can input "mi", "mile", or "miles".
  6. For yards, the user can input "yd", "yard", or "yards".
  7. For feet, the user can input "ft", "foot", or "feet".
  8. For inches, the user can input "in", "inch", or "inches".

Also, the types of measurement should be case insensitive. That means that "Meters", "METERS", and "mEtErS" should work correctly as well as "meters".

The output should contain full names of measurement types. Also, if the value is 1.0, the name of the measurement type must be singular. For example, "1.0 meter" but "1.1 meters".
